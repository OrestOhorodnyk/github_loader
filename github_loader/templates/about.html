GitHub Loader

Just a couple of steps to upload this app to your repository:

  1. Login with your GitHub credentials
  2. Click on the 'Create Repository' in the top left corner
  3. Enter name and description for new repository
  4. Click the 'Create Repository' button

It may take some time to load the code of the app to you repo. Once done you'll see success message. And you'll see new repository in the list of your repositories on home page
